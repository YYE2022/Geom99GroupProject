Problem Statement

Ideal

To alert the public of unsafe water quality for recreational purposes within Ontario provincial streams using a web map that can dynamically display the data.

Reality

Operational layers in the web map format are hard to understand regarding water quality.

Consequences

The web map doesn't properly display the data in the context of seeing water quality.

Proposal

Develop a web solution using multiple, web-based data sources to better display features of Ontario streams and lakes to public users.
