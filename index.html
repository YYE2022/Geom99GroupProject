Ontario Water Quality Map Problem Statement

The Ideal Solution

To provide an easy web map solution for the public to view unsafe water quality for multiple use cases within Ontario provincial streams where the web map can dynamically display the data.

The Reality

Operational layers in the web map format are hard to understand regarding water quality.

The Consequences

The web map doesn't properly display the data in the context of seeing water quality in a user friendly way. Public users can be confused with what is considered poor water quality

Our Proposal

Develop a web solution using multiple, web-based data sources to better display water quality features of Ontario streams and lakes to public users.

The Technology Explored

ArcGIS Developers, Esri Leaflet, ArcGIS Dashboard, ArcGIS Server and ArcGIS Online

Current Web Map

The current web map that is difficult to use in displaying water quality in streams.
